# Overview

유니티를 활용해 간단한 방탈출 컨셉의 PC 게임을 제작합니다.
게임에 필요한 리소스는 모두 제가 직접 제작하며 유니티를 활용한 게임 개발을 통 해 C#에 익숙해지고 게임 개발을 경험해보는 것이 목적입니다.
2D PC 게임을 제작하기 위해 가장 대중적인 게임개발 프로그램인 유니티를 사용하 였고 버튼과 패널을 주로 활용하여 개발했습니다.
